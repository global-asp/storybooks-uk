Ich kann mit ihm laufen.

I can run with it.

Ich kann mit ihm springen.

I can jump with it.

Ich kann mit ihm tanzen.

I can dance with it.

Ich kann mit ihm schwimmen.

I can swim with it.

Ich kann mit ihm hüpfen.

I can skip with it.

Ich kann mit ihm schießen.

I can kick with it.

Ich kann mit ihm weglaufen.

I can escape with it.

Aber ich kann niemals mit ihm fliegen.

But I can never fly with it.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Lisa Birkner
Read by: Jula Eberth
Language: German
Level: Level 1
